## Enunciado

Como sabemos, precisamos desenvolver um sistema web para o CTD Commerce.

O líder técnico, para esta etapa, pede que você disponibilize uma API para gerenciar as operações do produto.

A API deverá permitir:

    Registrar um novo Produto:
    Path: /Products
    Método: POST

Atualizar os dados de um produto existente:

    Path: /Products
    Método: PUT
